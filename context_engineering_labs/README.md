# Context Engineering Labs

4 labs, one per context engineering method, each drawing from all 4 project approaches:

| Lab | Method | Topics Covered |
|-----|--------|---------------|
| `lab_1_externalize.ipynb` | Write / Externalize | File-based KB, TF-IDF indexes, LLM knowledge cards |
| `lab_2_select.ipynb` | Select | Keyword strategies, TF-IDF selection, token importance scoring |
| `lab_3_isolate.ipynb` | Isolate | Per-user profiles, per-domain indexes, isolated method comparison |
| `lab_4_compress.ipynb` | Compress | Keyword threshold, TF-IDF filter, LLM multi-level compression |
